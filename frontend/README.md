# Vehicle Tracker

A real-time vehicle movement simulator using **React**, **React-Leaflet**, and **Tailwind CSS**.

## Features
- Full route (gray)
- Traveled path (red)
- Smooth animated car marker
- Play / Pause / Reset
- Live: Lat, Lng, Time, Speed
- Responsive UI

## Tech Stack
- React + Vite
- React-Leaflet
- Tailwind CSS
- OpenStreetMap

## Setup
```bash
npm install
npm run dev
